Cross Validation Report

The cross-validation process starts with a subset of data for model training, forecasts for a following period, and measuring the forecasting performance. The same forecasted period will then be added to the training set for the next iteration.

This cross-validation exercise has split data into 20 folds. Each fold contains 194 datapoints. The starting training set includes 10 folds. The testing set is fixed as 1 fold(s) and there are 10 iterations in total.

Cross Validation for the Conditional Mean and Example on Model Combination

RMSE is used to measure the predictive performance of the models in terms of conditional mean. In total, 8 models are selected for model combination according to the thresholds*.

Cross Validation for the Conditional Density and Example on Model Combination

MIS is used to measure the predictive performance of the models in terms of conditional density. In total 6 models are selected for model combination according to the thresholds*.
